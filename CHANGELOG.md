# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.0.1] - 2024-07-12

### Added
- Initial release of streamlit-sidebar-accordion-menu
- Support for accordion-style sidebar menus
- Automatic page navigation with `st.switch_page()`
- Dictionary format for easy menu definition
- Keyboard navigation support
- Accessibility features (ARIA attributes)
- Auto-detection of main script
- Support for nested menu structures
- React-based frontend with TypeScript
- Comprehensive documentation and examples
